RGB vs CMYK - Entendendo os Modelos de Cor

O que é RGB?

O modelo RGB (Red, Green, Blue) é um sistema de cores aditivas usado em dispositivos eletrônicos, como telas de computadores e celulares. As cores são formadas combinando luzes vermelha, verde e azul em diferentes intensidades.

Funcionamento do Pixel no RGB

Em telas digitais, cada pixel é composto por três subpixels: vermelho, verde e azul. A variação da intensidade dessas cores cria a percepção de uma ampla gama de cores.

O que é CMYK?

O modelo CMYK (Cyan, Magenta, Yellow, Black) é um sistema de cores subtrativas usado na impressão. As cores são criadas subtraindo luz refletida do papel, misturando tintas ciano, magenta, amarelo e preto.

RGB vs CMYK

RGB é ideal para telas, pois utiliza luz para formar cores vibrantes. CMYK é melhor para impressão, pois mistura tintas para obter cores precisas no papel. Ao converter imagens de RGB para CMYK, algumas cores podem ser alteradas devido às limitações do processo de impressão.
